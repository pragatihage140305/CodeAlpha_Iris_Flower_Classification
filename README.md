# Iris Flower Classification

## Project Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## Model Used
- Random Forest Classifier

## Accuracy
- 100%

## Features
- Data Visualization
- Classification Model
- Accuracy Evaluation
